# nodegraphql
Node server GraphQL example

# Introduction

Shows how to use Node with GraphQL 

# Usage
````
npm i
````
#
````
npm run dev
````
#
Browse to GraphQL IDE:

````
http:\\localhost:4000/graphql
````

#
Enter GraphQL such as:

````javascript
{
  pallets {
    id
    item
    quantity
  }
}
````
